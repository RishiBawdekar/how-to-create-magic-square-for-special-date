How To Create Magic Square For Special Date?

A magic square is a n x n square grid filled with distinct positive integers such that sum of all rows, all columns and two diagonals is equal. The EXCEL file uses mixed integer programming to create a magic square for a special date - like your birth date!

Prerequisite:

1. OpenSolver Excel Add-in. Read more here - Open Solver Home Page

OpenSolver will be available until you quit Excel. If you wish to make OpenSolver always available in Excel, the files from the zip all need to be copied into the Excel add-in directory. This is typically either:

a. `C:\Documents and Settings\”user name”\Application Data\Microsoft\Addins\`

b `C:\Users\"user name"\AppData\Roaming\Microsoft\Addins\`

Download .zip

After download:

1. Open Excel file.

2. Add OpenSolver as Excel Addin. Read more (applies to Excel 2010)

3. Type dates in cells B2 till E2. e.g. if today's date is 25/06/2016, then

  1. B2: 25
  2. C2: 6
  3. D2: 20
  4. E2: 16

4. Cells B2, C2, D2 and E2 are the ONLY CHANGEABLE CELLS IN EXCEL FILE. PLEASE DO NOT CHANGE OTHER CELLS.

5. Click on Solve.
